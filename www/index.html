🔧 API 端点一览

方法 路径 说明 Content-Type
GET / 或 /index.html 本页面（静态 HTML 文件服务） text/html 试试
GET /hello Hello 响应 text/plain 试试
GET /sleep/<ms> 延迟响应（测试并发隔离，上限 5000ms） text/plain 试试
GET /help API 帮助文本 text/plain 试试
GET /users 列出全部用户（BST 中序遍历） text/plain 试试
GET /users/<name> 按名称查找用户（链表遍历） text/plain 试试
GET /user/<name> 按名称查找用户（简化路径） text/plain 试试
GET /users/find-index/<name> 通过 RBT 索引查找用户 text/plain 试试
GET /users/compare/<name> 链表 vs RBT 搜索性能对比 text/plain 试试
POST /users 添加用户（body: CSV 行） text/plain —
DELETE /users/<name> 删除用户 text/plain —
